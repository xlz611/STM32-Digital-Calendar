# 基于 STM32 的数字式万年历

> 作者：GitHub：[@xlz611](https://github.com/xlz611)

一个以 **STM32F103C8T6** 为核心控制器的多功能数字式万年历系统，集成**时钟、日历、闹钟、秒表**四大功能，采用 DS1302 实时时钟芯片保证计时精度，8 位 LED 数码管动态扫描显示，4 个独立按键完成模式切换与参数校准，蜂鸣器实现闹钟声光提醒。

---

## 📌 项目简介

传统万年历功能单一、交互繁琐。本项目使用 STM32 单片机，将计时、日历、提醒、计时器整合为一体，适用于家庭作息管理、学习工作计时、运动赛事计时等场景。系统经方案选型，最终确定「STM32F103C8T6 + LED 数码管 + DS1302」方案，在性能、功能扩展性与成本之间达到平衡。

实测指标：
- 计时精度误差 **≤ 1 秒/天**（连续运行 24 小时实测误差 0.8 秒）
- 整机静态功耗 **≤ 60mA**
- 工作电压 **5V**（经稳压芯片转为 3.3V 供电）
- 断电后时间与闹钟参数不丢失（DS1302 纽扣电池备份）

---

## ✨ 功能特性

- 🕐 **时钟**：时/分/秒显示，24 小时制
- 📅 **日历**：年（2000–2099）/月/日，闰年自动识别（含 2 月 29 日处理）
- ⏰ **闹钟**：可设置闹钟时间，到点蜂鸣器声光提醒
- ⏱ **秒表**：启/停/复位，基于定时器中断实现 1 秒精确计时
- 🔘 **4 键交互**：模式切换 / 加 / 减 / 确认启停，软件消抖
- 💾 **掉电保护**：DS1302 备用电池维持时间数据
- 🔆 **强光可读**：8 位 LED 数码管替代 LCD，亮度高、响应快

## 🔧 硬件组成

| 模块 | 型号 / 说明 |
|------|------------|
| 主控 | STM32F103C8T6（ARM Cortex-M3，72MHz，64KB Flash，20KB SRAM） |
| 实时时钟 | DS1302（三线接口，CR2032 纽扣电池备份） |
| 显示 | 8 位共阴极 LED 数码管（74HC573 锁存器驱动，动态扫描） |
| 按键 | 4 × 独立按键（10KΩ 上拉，10ms 软件消抖） |
| 提醒 | 有源蜂鸣器 |
| 电源 | 5V DC → AMS1117-3.3V 稳压，0.1μF + 100μF 滤波 |
| 时钟源 | 8MHz 无源晶振 + 22pF 电容，经内部 PLL 倍频至 72MHz |

> ⚠️ **说明**：原课程设计文档个别章节残留 "LCD1602" 字样，系撰写时的笔误；**实际硬件与代码均为 8 位 LED 数码管**（见 `main.c` 中的 `LED_SD_Dpy` 动态扫描函数）。

## 📍 引脚分配

| STM32 引脚 | 外设 | 模式 |
|-----------|------|------|
| PB0–PB7 | LED 数码管 位选 | 推挽输出 |
| PC0–PC7 | LED 数码管 段选 | 推挽输出 |
| PB8–PB11 | 独立按键 KEY1–KEY4 | 上拉输入 |
| PB12 | 蜂鸣器 | 推挽输出 |
| PB13 | DS1302 SCLK | 双向 I/O |
| PB14 | DS1302 I/O | 双向 I/O |
| PB15 | DS1302 CE | 双向 I/O |
| PA0 / PA1 | 8MHz 外部晶振（OSC_IN / OSC_OUT） | 模拟 |

## 🧩 软件架构

采用 C 语言**模块化编程**，主要功能模块：

| 模块 | 职责 |
|------|------|
| 初始化模块 | GPIO / 定时器 / 中断 / DS1302 / 数码管 / 蜂鸣器初始化 |
| 时间读取与处理 | 通过三线接口读取 DS1302（BCD 码），转十进制；闰年判断；秒表计时 |
| 显示控制 | 十进制数据 → 段码，动态扫描多模式显示 |
| 按键处理与校准 | 查询方式检测按键，10ms 消抖，模式切换与参数加减 |
| 闹钟 / 秒表控制 | 闹钟比对触发蜂鸣器；TIM3 中断驱动秒表 1 秒计时 |

**定时器**：TIM3 配置为 10KHz 计数、周期 1000，产生 **1 秒** 定时中断，用于时钟秒数递增与秒表计时。

## 🛠 开发环境与编译

- **IDE**：Keil MDK5（μVision）
- **库**：STM32F1xx 标准外设库（Standard Peripheral Library）
- **编译器**：ARMCC
- **烧录**：ST-Link / 串口 ISP（如 FlyMCU）

**步骤**：
1. 用 Keil MDK5 新建 STM32F103C8T6 工程；
2. 加入 `main.c` 及上述外设驱动源文件；
3. 配置芯片型号、时钟（HSE 8MHz → PLL 72MHz）、下载器；
4. 编译并烧录到开发板；
5. 上电后默认进入时钟模式，按 KEY4 循环切换功能。

## ▶️ 运行效果（实测）

| 模式 | 显示格式 | 示例 |
|------|---------|------|
| 时钟 | `HH:MM:SS` | `14:30:25` |
| 日历 | `YYYY-MM-DD` | `2024-06-15` |
| 闹钟 | `AL:HH:MM` | `AL:07:00` |
| 秒表 | `ST:HH:MM:SS` | `ST:00:01:23` |

- 连续运行 24 小时时间误差 **0.8 秒**，满足精度要求；
- 秒表计时 1 分 23 秒后停止显示 `ST:00:01:23`，长按 2 秒复位为 `ST:00:00:00`；
- 闰年（2024-02-29）日期显示正常。

## 🚧 已知问题与改进方向

**调试中解决的问题**
- 数码管显示乱码/闪烁 → 调整动态扫描延时至 1ms，确认段/位选引脚均为推挽输出；
- 按键误触发 → 加入 10ms 软件消抖，上拉电阻换为 10KΩ；
- 秒表不准 → 核对 TIM3 参数（10KHz / 周期 1000），提升定时器中断优先级。

**可扩展方向**
- 增加农历显示、多组闹钟、温度/湿度（DHT11）检测；
- 显示升级为 LCD12864 或加亮度自动调节；
- 集成 ESP8266（Wi-Fi）/ 蓝牙，支持 NTP 网络校时与手机远程配置。
